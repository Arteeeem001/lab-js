Практична робота №11–12: REST API та CRUD
Студент: Геворгян Артем
Проєкт: Адмін-панель та каталог із живою взаємодією з сервером.

Реалізовано:
CRUD-операції: Повний цикл роботи з даними (Створення, Читання, Оновлення, Видалення) через fetch().

API-шар: Логіка запитів винесена в окремий модуль api.js.

Динамічний інтерфейс: Автоматичне оновлення каталогу після додавання або видалення записів без перезавантаження сторінки.

Візуалізація: Картки послуг із підтримкою зображень та категорій.

Запуск:
Сервер (Back-end): json-server --watch db.json --port 3000

Сайт (Front-end): python -m http.server 8000
